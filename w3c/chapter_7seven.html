文本域(Text fields)
本例演示如何在 HTML 页面创建文本域。用户可以在文本域中写入文本。

名:
姓:

密码域
本例演示如何创建 HTML 的密码域。

用户: 密码:

请注意，当您在密码域中键入字符时，浏览器将使用项目符号来代替这些字符。

复选框
本例演示如何在 HTML 页中创建复选框。用户可以选中或取消选取复选框。

我喜欢自行车: 我喜欢汽车:

单选按钮
本例演示如何在 HTML 中创建单选按钮。

男性: 女性:

当用户点击一个单选按钮时，该按钮会变为选中状态，其他所有按钮会变为非选中状态。

简单的下拉列表
本例演示如何在 HTML 页面中创建简单的下拉列表框。下拉列表框是一个可选列表。

文本域(Textarea)
本例演示如何创建一个文本域（多行文本输入控件）。用户可以在文本域中写入文本。在文本域中，可写入的字符字数不受限制。

您无法对本例进行编辑，因为我们的在线编辑器使用 textarea 进行输入， 而浏览器不允许 textarea 中存在另一个 textarea。

本例演示如何创建按钮。你可以对按钮上的文字进行自定义。
